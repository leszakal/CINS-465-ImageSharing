About

- Site Purpose -

This is an image sharing site that allows users to upload photos or artwork with custom titles, descriptions, and tags. Posts can be edited or removed as Users please. Depending on user choice, posts can be visible to any visitor to the site, or only to those with accounts. Registered users to interact with posts via the comment system. Be sure to utitlize the basic search function to find posts based on title or tags.
